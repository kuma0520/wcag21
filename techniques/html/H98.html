HTML 5.2 autocomplete 属性の使用

達成方法に関する重要な情報

これらの達成方法 (参考) の使用法及び、それらが WCAG 2.1 達成基準 (規定) とどのように関係するかに関する重要な情報については、WCAG 達成基準の達成方法を理解するを参照のこと。適用 (対象) セクションは、その達成方法の範囲について説明しており、特定の技術に関する達成方法があるからといって、WCAG 2.1 を満たすコンテンツを作成する際に、常にその技術が使用可能であるわけではない。

適用 (対象)

HTML 5.2 autofill トークンに対応するすべての HTML フォームフィールド。

これは、達成基準 1.3.5: 入力目的の特定（十分）に関連する達成方法である。

解説

この達成方法の目的は、入力が機械でも解釈できるように、事前に定義及び公開された分類用語をプログラム的に入力へリンクすることである。この方法では、どの言語が使用されているか、又は入力にラベルを付けるためにどういった画面上に表示される用語が使用されているかに関係なく、入力には常に共通で、参照可能及び識別可能な関連付けられた値を持つ。その後、さらにユーザーを支援するためにカスタマイズ又は機械操作することができる。

この方法では、フォームの各フォームフィールドに適切な autocomplete トークンを追加して、特定の入力をプログラムで確定できる。これは、制作者が使用しているラベルが馴染みがないためにフィールドの意味をすぐには理解できない認知障害の人に役立つ。入力がプログラムによって割り当てられている場合、サードパーティのプラグイン及びソフトウェアがこれらのフォームフィールドを操作して、認知障害のあるユーザーがアクセスしやすくすることができる。例えば、プラグインはテキスト文字列 "tel" を持つ autocomplete トークンを検出し、電話アイコンを挿入することができる。さらに、サードパーティのソフトウェアが、既存のラベルをユーザーがよく知っているテキストラベルと交換できるようになる。たとえば、"Given Name" を "First Name" に変更できる。

autocomplete 属性を使用すると、ユーザーが望む値をフォームフィールドに事前に入力するブラウザの機能も向上する。入力が既知のトークン値で適切に 'タグ付け' されると、ユーザーが入力した値はホストマシン上のローカルに保管され、再利用のためにトークン値に関連付けられる。これは、タイピングに問題をもつ器用さの障害を持つ人、より多くの時間を必要とするかもしれない人、そしてフォームに記入するための労力を減らしたいと望む人に役立つ。

autocomplete属性を使用すると、ブラウザにローカルで格納されている値のリストに対してパターンマッチを実行でき、入力がプログラムによりタグ付けされたときに適切な一致する値を提供できます。これは、エンドユーザーがオン又はオフにしたり、変更したりするユーザー設定です。これにより、格納された値を使用してフィールドに値を入力するため、タイピングが減り、記憶への依存が少なくなります。

セキュリティへの配慮

組織としては、入力フィールドを自動で埋められるようにすることに不安を感じることがあります。 ブラウザがどのように情報を保存しているのか、セキュリティはどうなっているのか、という点で混乱することがあります。 の意味合いがあります。

autocomplete 属性の場合:

  • このテクニックは、フォームに記入しているユーザーのデータを尋ねる場合にのみ使用し、他の人のデータを尋ねる場合には使用しないでください。
  • これは、同じコンピュータ上で同じユーザーアカウントを使用していて、かつ 同じブラウザを使用しています。マルチログインのシナリオでは 異なるアカウントを使用しています。(ユーザーはコンピュータ間でデータの同期を設定することができますが、それは デフォルトではありません)。
  • オートコンプリートによる情報の保存は、ユーザーによるオプトインで、通常は 最初にデータを保存します。
  • フォームは自動送信ではなく、ユーザーは送信前のデータを見ることができます。
  • ブラウザの設定で履歴もフォームデータも簡単に消去できます。
  • プライベートブラウジングなどのプライバシーモードを簡単にかけることができます。
  • ウェブページにオートコンプリートが設定されていなくても、ブラウザはデータを保存することができますし、一部のプラグイン（例えば パスワード管理者) は積極的にヒューリスティックを使用して、どのフィールドが何のためのものかを推測します。 記入してください。オートコンプリート属性を使用すると、これらの推測が正確になります。

ブラウザの履歴は、人々が何をしたかについてはるかに詳細な情報を提供してくれます。 オートコンプリートデータとして利用可能です。ブラウザ履歴の解決策/軽減策は オートコンプリートに似ています。

  </div>

  </div>
</form>

  2. ラベルで示されたフォームフィールドの目的は、オートコンプリートに対応しています。 トークンを入力します。
